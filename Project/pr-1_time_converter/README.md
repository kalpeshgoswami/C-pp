⏱️ Time Converter (C++)

A simple C++ console application that converts time between seconds and HH:MM:SS format using a class-based approach.

📌 Features

Convert total seconds → hours : minutes : seconds

Convert hours : minutes : seconds → total seconds

Menu-driven program

Beginner-friendly C++ code using classes

🛠️ Technologies Used

Language: C++

Compiler: GCC / g++

Concepts: Classes, Functions, Switch Case, Arithmetic Operations

📂 Project Structure
TimeConverter/
│
├── main.cpp
└── README.md

▶️ How to Compile and Run
Using g++ (Linux / macOS / Windows with MinGW)
g++ main.cpp -o TimeConverter
./TimeConverter
